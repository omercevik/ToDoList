Welcome to To-Do Lists!


This project is about to create to-do lists.


  • You can register to system.
  • You can login to system.
  • You can create a To-Do List category as lists.
  • You can create To-Do List items for each To-Do List.
  • You can order the To-Do Lists by Item Name clicking on 'Item Name' button on table.
  • You can order the To-Do Lists by Created Date clicking on 'Created Date' button on table.
  • You can order the To-Do Lists by Deadline clicking on 'Deadline' button on table.
  • You can order the To-Do Lists by Status clicking on 'Status' button on table.
  • You can delete To-Do List items.
  • You can delete To-Do List.
